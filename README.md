# C Learning Workspace

This repository is organized as a simple, beginner-friendly C practice workspace.

## Learning Structure

- basics: Basic input and output
- conditionals: if-else based decision making
- loops: Iteration with practical examples
- functions_recursion: Reusable functions and recursion
- arrays_strings: Arrays and string operations
- file_io: Basic file reading

## Recommended Study Order

1. basics/01_hello_world.c
2. basics/02_sum_two_numbers.c
3. conditionals/01_even_or_odd.c
4. conditionals/02_grade_from_marks.c
5. loops/01_fibonacci_iterative.c
6. loops/02_prime_check.c
7. functions_recursion/01_factorial_recursive.c
8. functions_recursion/02_fibonacci_recursive.c
9. arrays_strings/01_palindrome_string.c
10. arrays_strings/02_count_nonzero_2d_array.c
11. file_io/01_read_file_line_by_line.c

## Compile and Run

From the project root, compile any file with gcc:

```bash
gcc basics/01_hello_world.c -o hello
./hello
```

Another example:

```bash
gcc functions_recursion/01_factorial_recursive.c -o factorial
./factorial
```

## Notes

- This repo is intentionally flat and easy to browse.
- Each folder teaches one topic.
- Start from basics and move down the list.

## Suggested Practice Method

1. Run the file as-is.
2. Modify one condition or input.
3. Predict output before running again.
4. Add a new test case.
5. Create your own variation in the same folder.
